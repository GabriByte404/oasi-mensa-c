# Oasi Mensa C
Simulazione multiprocesso di una mensa automatizzata in ambiente UNIX.

## Dettagli Tecnici
- Gestione della concorrenza tramite **Semafori System V**.
- Memoria condivisa (**Shared Memory**) per lo stato globale.
- Gestione dei segnali per la pulizia delle risorse IPC all'uscita.
